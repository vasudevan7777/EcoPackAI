MODULE 1: DATA COLLECTION AND DATABASE INTEGRATION
==================================================

Overview:
This module handles the initial data collection and storage in a MySQL database.
The system collects sustainable packaging materials data and stores it for further processing.

Components:
-----------
1. Database Schema (database_schema.sql)
   - Creates tables for materials and product categories
   - Sets up proper data types and constraints
   - Establishes relationships between tables

2. Data Loading Script (load_data.py)
   - Reads CSV files from the data folder
   - Connects to MySQL database
   - Inserts records into appropriate tables
   - Handles data validation and error checking

3. Data Viewing Script (view_data.py)
   - Retrieves data from the database
   - Displays materials and categories
   - Provides quick data verification

Database Tables:
----------------
Materials Table:
- material_id: Unique identifier
- material_name: Name of the packaging material
- material_type: Category (Paper, Plastic, Bio-based, etc.)
- strength_mpa: Tensile strength in megapascals
- weight_capacity_kg: Maximum weight capacity
- biodegradability_score: Environmental degradation rate
- co2_emission_score: Carbon footprint metric
- recyclability_percent: Percentage of recyclable content
- cost_per_kg: Price per kilogram

Product Categories Table:
- category_id: Unique identifier
- category_name: Product category
- required_strength: Minimum strength requirement
- typical_weight: Standard weight for category

Data Sources:
-------------
- materials_data_100_records.csv (100 material samples)
- product_categories_100_records.csv (Product specifications)

How to Use:
-----------
1. Set up MySQL database with credentials
2. Run database_schema.sql to create tables
3. Execute load_data.py to import data
4. Use view_data.py to verify data was loaded correctly

Expected Output:
----------------
- Database with populated materials and categories
- Data ready for processing in Module 2

Notes:
------
This module establishes the foundation for the entire system by organizing
raw material data into a structured database format.
